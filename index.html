Substring is similar to slice, but it can't accept negative values.
let str = "pie, cake, apples"
let res = str.substring(7,13);


Substr is similar to slice, but the second parameter specifices the lenthg of the extracted part.
var str = " apple walnut banana"
var res = str.substr(7, 16);


When you omit the second string, substr simply splices out the rest of the string.
var str = "apple banana walnut"
var res = str.substr(7);


If the first paraemter is negative, then the position simply counts from the end of the string.
var str = "apple banana walnut"
var res = str.substr(-4);
